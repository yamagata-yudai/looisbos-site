# Looisbos Official Website

Official website for the band **Looisbos**.
Built with HTML, CSS, and Vanilla JavaScript.

## Project Structure
- `index.html`: Main page
- `style.css`: Styling (Dark mode, animations)
- `script.js`: Interactions (Mouse cursor, Mobile menu)
- `logo.png`: Band logo
- `artist.jpg`: Hero background image

## Deployment
This site is designed to be hosted on **GitHub Pages**.

1. Go to Settings > Pages
2. Select `main` branch as source
3. Save
